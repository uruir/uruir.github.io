Markdown 简介

发表于 12/16/2013 | 修改于 12/10/2015   |   分类于 工具   |  

以下简介翻译自Markdown Site:

Markdown 是一个 web 作家的文本到 HTML 转换工具。它能让你以易读、易写的文本格式（即以.md为后缀的文件）来写作，然后将文件转换成有效的结构化标记语言（类 HTML 文件）。

这样，”Markdown”具有以下两点特性：

  1. 纯文本格式的语法
  2. 一个用 Perl 语言编写的，能将文本格式转为 HTML 的转换工具

详细的语法说明，请点击这里。

Markdown 语法小玩意儿，请点击这里。

Markdown 语法格式的终极要义是：具有最大化的可读性！也就是用 Markdown 写出的文档应该是以它做为纯文本的本来面目显示出来，而不是看起来像标记语言那样具有标签或版面格式。在 Markdown 问世过程中，有受其它文本到标记过滤器的影响，但影响最大的是纯文本邮件。
